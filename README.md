Описание проекта
Это графическое приложение (GUI) для генерации криптографически стойких паролей. Программа позволяет пользователю гибко настраивать параметры будущего пароля: его длину и используемые наборы символов (цифры, буквы, спецсимволы). Все сгенерированные пароли сохраняются в историю, которая доступна прямо в интерфейсе.

Проект реализован на языке Python с использованием библиотеки customtkinter для создания современного и удобного интерфейса.

Технологии и зависимости
Язык программирования: Python 3.9+
Библиотеки:
customtkinter — для создания графического интерфейса.
random (стандартная библиотека) — для генерации случайных последовательностей.
json (стандартная библиотека) — для сохранения истории паролей в файл.

Инструкция по использованию
Настройка длины: Используйте ползунок «Длина пароля», чтобы задать желаемое количество символов (от 4 до 32).
Выбор символов: Отметьте галочками чекбоксы, чтобы включить нужные типы символов:
Цифры (0-9)
Буквы (a-z, A-Z)
Спецсимволы (!@#$%^&*...)
Генерация: Нажмите кнопку «Сгенерировать пароль».
Результат: Сгенерированный пароль появится в поле для чтения. Он также будет автоматически добавлен в таблицу «История» внизу окна.
Структура проекта

README.md — этот файл, описание проекта.
